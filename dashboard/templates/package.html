📦 Forensic Evidence Package

📋 Package Contents

The evidence package is a cryptographically verifiable ZIP archive containing complete chain-of-custody documentation. It is designed to be submitted as digital forensic evidence.

  • 📄 evidence_records.json

    All access decisions with full context: user, resource, action, IP address, risk score, and policy reason.

  • 🔗 hash_chain_verification.json

    Per-record hash verification results showing stored vs computed hashes — instantly reveals any tampering.

  • 📋 chain_of_custody.json

    Package metadata: generation timestamp, system info, total records, and a SHA-256 hash of the entire evidence set.

  • 📊 forensic_report.html

    Self-contained HTML report with executive summary, evidence tables, hash verification status, and methodology section. Open in any browser — no internet needed.
